# AI Research Assistant

A multi-source RAG-based research assistant built with Python and OpenAI.

## Setup
1. Create virtual environment: `python -m venv venv`
2. Activate: `source venv/bin/activate` (Mac/Linux) or `.\.venv\Scripts\activate.ps1` (Windows-PowerShell) or `.\.venv\Scripts\activate.bat` (Windows-Command Prompt)
3. Install dependencies: `uv`
4. Copy `.env.example` to `.env` and add your API keys

## Project Structure
- `src/` - Source code
- `notebooks/` - Jupyter notebooks for experiments
- `data/` - Data files (PDFs, etc.)
- `tests/` - Test files
